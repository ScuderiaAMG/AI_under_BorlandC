Borland C++ 是一款经典的 C 和 C++ 集成开发环境（IDE），以下为你介绍其基本使用方法：

### 安装与启动
- **安装**：获取 Borland C++ 的安装程序，按照安装向导的提示进行操作。在安装过程中，你可以选择安装路径和所需的组件。安装完成后，桌面上会出现 Borland C++ 的快捷图标。
- **启动**：双击桌面上的 Borland C++ 快捷图标，或者通过开始菜单找到并启动该程序。启动后，你将看到 Borland C++ 的主界面。

### 创建新项目或源文件
- **创建新项目**
    - 打开 Borland C++ 后，选择“File（文件）”菜单，然后点击“New（新建）”。
    - 在弹出的对话框中，选择“Project（项目）”选项卡。根据你的需求选择合适的项目类型，如“Console Application（控制台应用程序）”用于创建简单的命令行程序。
    - 为项目指定一个名称和保存路径，然后点击“OK（确定）”。Borland C++ 会自动创建项目的基本结构和相关文件。
- **创建源文件**
    - 如果不需要创建项目，只是编写单个源文件，可以选择“File（文件）”菜单，然后点击“New（新建）”。
    - 在弹出的对话框中，选择“Source File（源文件）”选项卡，文件类型选择“C++ Source File（C++ 源文件）”或“C Source File（C 源文件）”，取决于你要编写的代码是 C 还是 C++。
    - 为源文件指定一个名称和保存路径，然后点击“OK（确定）”。此时会打开一个空白的代码编辑窗口。

### 编写代码
- 在代码编辑窗口中输入 C 或 C++ 代码。例如，以下是一个简单的 C++ 控制台程序示例：
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, Borland C++!" << endl;
    return 0;
}
```
- 在编写代码时，要注意语法规则和代码的规范性。Borland C++ 提供了代码高亮显示功能，不同类型的代码会以不同的颜色显示，方便你区分和阅读。

### 编译与运行
- **编译代码**
    - 编写完代码后，选择“Compile（编译）”菜单，然后点击“Compile（编译）”选项（快捷键通常是 F9）。Borland C++ 会对代码进行语法检查和编译，如果代码中存在语法错误，会在“Message（消息）”窗口中显示错误信息，你需要根据错误信息修改代码。
    - 如果代码编译成功，会生成可执行文件。可执行文件的名称通常与项目或源文件的名称相同，扩展名为.exe。
- **运行程序**
    - 编译成功后，选择“Run（运行）”菜单，然后点击“Run（运行）”选项（快捷键通常是 Ctrl + F9）。程序会在控制台窗口中运行，并显示输出结果。
    - 如果程序需要输入数据，你可以在控制台窗口中输入相应的数据，然后按回车键确认。

### 调试程序
- **设置断点**：在代码编辑窗口中，将光标定位到需要设置断点的代码行，然后选择“Debug（调试）”菜单，点击“Toggle Breakpoint（切换断点）”选项（快捷键通常是 F5）。断点处会出现一个红色圆点，表示程序运行到此处会暂停。
- **开始调试**：选择“Debug（调试）”菜单，点击“Start Debugging（开始调试）”选项（快捷键通常是 Ctrl + F7）。程序会开始运行，当遇到断点时会暂停。
- **单步执行**：在调试过程中，你可以使用“Step Over（单步跳过）”（快捷键 F8）、“Step Into（单步进入）”（快捷键 F7）等命令来逐行执行代码，观察程序的执行流程和变量的值。通过调试可以帮助你找出代码中的逻辑错误。

### 其他常用操作
- **保存文件**：在编写代码过程中，要经常选择“File（文件）”菜单，点击“Save（保存）”选项（快捷键通常是 Ctrl + S）保存文件，以免数据丢失。
- **打开已有文件**：如果需要打开之前编写的代码文件，选择“File（文件）”菜单，点击“Open（打开）”选项，然后在弹出的对话框中选择要打开的文件。
